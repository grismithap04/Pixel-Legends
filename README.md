# Pixel Legends — Game Character Store

A front-end web project built with HTML, CSS, and vanilla JavaScript.
Pixel Legends is a fictional game character store where users can browse,
purchase, and manage orders for pixel-art game characters.

## Features

- Full-screen hero section with a looping background video
- Character grid showcasing 7 unique fighters across different types
  (Tank, Rogue, Brawler, Archer, Warrior, Adventurer)
- Purchase modal with buyer name input, quantity selector, and live total calculation
- Slide-in orders panel to review, remove, and confirm all selected characters
- Smooth scroll-based active navigation highlighting
- Floating pill-style navbar with blur backdrop
- Custom pixel font (Astoz) paired with DM Sans for a clean dark aesthetic
- Fully responsive across desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, animations, responsive breakpoints)
- Vanilla JavaScript (DOM manipulation, event handling, dynamic rendering)

## Note

This is a replica/front-end concept project — no backend or payment integration.
